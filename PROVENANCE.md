# Provenance and Attribution

RAGHub is a curated directory of Retrieval-Augmented Generation (RAG) tools, frameworks, projects, and resources.

## Current provenance status

This repository currently contains an MIT licence file with copyright attribution to Andrew Jang. That attribution should be preserved unless repository history or upstream project records clearly support a different change.

Maintainers and contributors should not treat repository ownership, a fork, a copy, or an account transfer as proof of original authorship.

## Directory status

RAGHub is an informational directory. Inclusion in the directory does not mean that the listed tool, framework, project, service, paper, or resource has been:

- endorsed by the maintainers
- security reviewed
- production validated
- benchmarked against alternatives
- checked for legal, privacy, or compliance suitability
- approved for use in any particular system

Readers should perform their own assessment before adopting any listed resource.

## Attribution practice

When adding or updating entries:

- preserve existing licence notices
- link to original source pages where possible
- avoid implying authorship, affiliation, endorsement, or partnership without evidence
- keep descriptions factual and neutral
- do not copy long marketing text from vendor sites

## Maintenance note

Star counts, activity indicators, project status, pricing, licences, and product availability can change quickly. Contributors should verify metadata before submitting changes, and readers should treat listed information as a starting point for research rather than a final source of truth.
